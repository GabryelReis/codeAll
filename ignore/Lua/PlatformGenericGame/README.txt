Esse é um simples projeto de tentativa para criar um jogo genérico de plataforma. Você pode utilizar esse
projeto para estudos e jogar esse jogo a vontade, os requisitos mínimos para jogá-lo é ter em seu computador
o interpretador Lua e o framework Love2D (LÖVE). Agradeço o DevJeeper, o CriaJogo, o Sheepolution e ao Augusto Manzano pelos tutoriais deles, foram de grande ajuda para mim.

Nesse momento, o jogo não está finalizado, pois ainda estou estudando a linguagem de programação Lua.

This is a simple attempt project to create a generic platform game. You can use this project for studies and 
play this game at will, the minimum requirements to play it is to have on your computer the Lua interpreter and the Love2D framework (LÖVE). I thank DevJeeper, the Cria Jogo Sheepolution and the Augusto Manzano for their tutorials, were of great help
to me.

At this time, the game is not finished, because I'm still studying the lua programming language.

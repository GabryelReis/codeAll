# Lua Aprendizado #

Por que a linguagem Lua? Lua é uma linguagem de programação brasileira, criada na PUC-RIO, estou dando  preferência a ela porque quero criar jogos utilizando o framework Love2D (LÖVE), que está disponível somente nessa linguagem. Java é uma linguagem legal, mas Lua é mais veloz, mais fácil e muito usada em jogos, uma área que eu desejo seguir.
